UVA 11087 - Divisibility Testing

题目描述：
给定 n 个整数 < a1 a2 a3 ... an > 和一个整数 k。找出选择 2 个整数 (ai, aj) 的方法数，
满足以下条件：
- ai ≤ aj
- 1 ≤ i, j ≤ n 且 i ≠ j
- (ai + aj) 能被 k 整除
- 每对必须不同

输入格式：
第一行包含整数 T，表示测试用例数量。
每个测试用例包含两行：
- 第一行：两个整数 n 和 k
- 第二行：n 个整数 a1, a2, ..., an

输出格式：
对于每个测试用例，输出案例编号和满足条件的配对数。

约束条件：
- T < 100
- 1 < n < 100001
- 0 < k < 501
- |ai| < 10000001

样例输入：
2
5 1
4 1 2 2 3
5 2
4 1 2 2 3

样例输出：
Case 1: 7
Case 2: 3
