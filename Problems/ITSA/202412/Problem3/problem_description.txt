Problem 3. 傳回紙條

(Time Limit: 1 seconds)

問題描述：
薩華與堅哥在早自習中互傳紙條，為了怕中途被頭皮的同學公開紙條的內容，他們偷偷發展了一種只有他們知道的簡單加解密方式，以保密紙條內容。

解密的方式如下：
1.字串只有小寫英文與空白字元。
2.第一個字元有意義。
3.接下來每間隔 N 個字元才有意義，其他都沒有意義。
4.N 為每天第一個紙條上，寫上 N。

輸入說明：
第一行為 1 個正整數 N (1≤N≤5)，代表相隔 N 個字元。
接下來唯一串字串(包含空白)，表示加密過的訊息。

輸出說明：
根據解密方式，輸出加密前的正確訊息。
每行輸出最後必有換行符號。

範例：
Sample1 Input:
2
las eelxcowhvcneop sflvbazevuiakp

Sample1 Output:
I love Java
