UVA 100 - The 3n + 1 problem

問題描述：
給定兩個整數 i 和 j，找出在 i 到 j 之間（包含 i 和 j）的所有整數中，
哪一個整數的 cycle length 最大。

Cycle length 的定義：
對於任何正整數 n，重複執行以下規則：
- 如果 n 是奇數，則 n = 3n + 1
- 如果 n 是偶數，則 n = n / 2
- 重複直到 n = 1

從開始到結束（包含 1）的步驟數就是 cycle length。

輸入格式：
每行包含兩個整數 i 和 j (1 ≤ i, j ≤ 1,000,000)

輸出格式：
對於每組輸入，輸出一行包含三個整數：
原始的 i、原始的 j、以及在 min(i,j) 到 max(i,j) 範圍內的最大 cycle length

範例：
輸入：
1 10
100 200
201 210
900 1000

輸出：
1 10 20
100 200 125
201 210 89
900 1000 174
